# TCP Chat App

A simple, real-time terminal chat application built with **Node.js**

## Features -
* **Real-time Messaging**: Instant communication between clients.
* **User Registration**: Your first message becomes your username.
* **Private Chat**: Send secret messages using `@username message`.
* **Broadcast**: Messages are sent to everyone except the sender.

## Run Locally

1. Clone the repository

```bash
git clone https://github.com/SupriyoP09/tcp-server.git
```

2. Go to the project folder

```bash
cd tcp-server
```

3. Start the server

```bash
node server.js
```

4. Open another terminal and run the client

```bash
node client.js
```
